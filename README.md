# 香燈執事 · 各法會 SOP

寶嚴僧團「香燈（心燈）執事」各法會標準作業流程大字看板。
**100% 依據寶嚴 Google 雲端硬碟內的實際文件**（司儀稿・儀軌版位・流程表・各式表單・工作流程・香燈課程教材）萃取；
查無之項目一律留空或標「缺資料」，不以常識杜撰，每條步驟皆標明出處。

**線上看板：** https://twnyda07.github.io/xiangdeng-sop/

## 內含（17 法會／活動）
- **完整 SOP**：盂蘭盆梁皇寶懺（含大蒙山）、往生佛事、楞嚴禪七、超薦法會、剃度、八關齋戒暨三昧水懺、結夏安居、香燈實務通則（佛事篇）、香燈物料標準（物料篇）
- **片段線索**：新春上燈、新眾講習
- **僅影音／曲譜**：華嚴、普門品、梵唄香燈總論、牌位／梵唄教材
- **空資料夾（待補件）**：清明地藏、浴佛節
- 另含香燈通則 18 條、資料覆蓋報告、建議補件清單。

## 使用
- 頂部**目錄**可跳到各法會；**搜尋框**打「往生／助念／施食／牌位／灑淨」等即時篩選。
- **顯示出處**鈕可逐條顯示〔來源檔名〕；點法會標題展開**時序時間軸**與壇場／供品／法器。
